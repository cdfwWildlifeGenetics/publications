# A genome assembly of the North American golden eagle, *Aquila chrysaetos canadensis*

Dataset DOI: [10.5061/dryad.2280gb65r](https://doi.org/10.5061/dryad.2280gb65r)

## Description of the data and file structure

This dataset contains the North American golden eagle genome assembly liftover annotation as well as accompanying code for generating the annotation and analytical code for comparing assembly quality to other Accipitridae assemblies.

### Data Files

**1. chroms.txt** -- list of syntenic scaffolds between European and CCGP North American golden eagle genome assemblies

**2. bAquChr2.0.hap1.liftover_polished.gtf** -- liftover annotation of the assembly using the European golden eagle reference genome (bAquChr1.4) annotation; generated by the script **run_liftoff.sbatch**


### Analysis Script Files

**1. CCGP_busco.sbatch** -- code for performing the BUSCO analysis; [slurm](https://slurm.schedmd.com/man_index.html) submission script

**2. run_minimap2.sh** -- code for aligning the CCGP North American golden eagle assembly ([bAquChr2.0.hap1](https://www.ncbi.nlm.nih.gov/datasets/genome/GCA_030820615.1/)) to the European golden eagle reference genome ([bAquChr1.4](https://www.ncbi.nlm.nih.gov/datasets/genome/GCF_900496995.4/)) for synteny assessment before performing liftover annotation (**run_liftoff.sbatch**); catenate script and execute each command on the command line

**3. run_liftoff.sbatch** -- code for generating the liftover annotation; [slurm](https://slurm.schedmd.com/man_index.html) submission script

## Code/software Requirements

**CCGP_busco.sbatch**

Dependencies: [BUSCO](https://busco.ezlab.org/) v5.0.0+

Inputs:

* aves_odb10 -- ortholog database; download to working directory using `busco download -l aves_odb10`
* directory containing all genome assemblies for generating BUSCO scores
  * [bAquChr2.0.hap1](https://www.ncbi.nlm.nih.gov/datasets/genome/GCA_030820615.1/) -- North American golden eagle (CCGP assembly)
  * [Aquila_chrysaetos-1.0.2](https://www.ncbi.nlm.nih.gov/datasets/genome/GCF_000766835.1/) -- North American golden eagle (2014 Van Den Bussche assembly)
  * [AquilaChrysaetos1](https://www.ncbi.nlm.nih.gov/datasets/genome/GCA_000696035.1/) -- North American golden eagle (2014 Doyle assembly)
  * [bAquChr1.4](https://www.ncbi.nlm.nih.gov/datasets/genome/GCF_900496995.4/) -- European golden eagle
  * [Ajapo_1](https://www.ncbi.nlm.nih.gov/datasets/genome/GCA_030162315.1/) -- Japanese golden eagle
  * [bHalAlb1.1](https://www.ncbi.nlm.nih.gov/datasets/genome/GCF_947461875.1/) -- White-tailed eagle
  * [bGypBar2.pri](https://www.ncbi.nlm.nih.gov/datasets/genome/GCA_028022735.1/) -- Bearded vulture
  * [bHarHar1_primary_haplotype](https://www.ncbi.nlm.nih.gov/datasets/genome/GCF_026419915.1/) -- Harpy eagle
  * [ASM2749777v1](https://www.ncbi.nlm.nih.gov/datasets/genome/GCA_027497775.1/) -- Black-mantled goshawk

**run_minimap2.sh**

Dependencies: [minimap2](https://github.com/lh3/minimap2#map-long-genomic) v2.29+, [samtools](https://www.htslib.org/doc/samtools.html) v1.17+

Inputs: reference genome fasta `${ref}` and output directory `${out}`

**run_liftoff.sbatch**

Dependencies: [liftoff](https://github.com/agshumate/Liftoff) v1.6.3+

Inputs: 
* **chroms.txt** -- list of syntenic scaffolds between Euorpean and CCGP North American golde eagle assemblies
* European golden eagle [reference assembly](https://ftp.ncbi.nlm.nih.gov/genomes/all/GCF/900/496/995/GCF_900496995.4_bAquChr1.4/GCF_900496995.4_bAquChr1.4_genomic.fna.gz) and [genome annotation](https://ftp.ncbi.nlm.nih.gov/genomes/all/GCF/900/496/995/GCF_900496995.4_bAquChr1.4/GCF_900496995.4_bAquChr1.4_genomic.gtf)
* CCGP North American golden eagle [genome assembly](https://ftp.ncbi.nlm.nih.gov/genomes/all/GCA/030/820/615/GCA_030820615.1_bAquChr2.0.hap1/GCA_030820615.1_bAquChr2.0.hap1_genomic.fna.gz)
