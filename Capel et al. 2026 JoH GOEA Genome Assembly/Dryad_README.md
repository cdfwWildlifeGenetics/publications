# A genome assembly of the North American golden eagle, Aquila chrysaetos canadensis

Dataset DOI: [10.5061/dryad.2280gb65r](https://doi.org/10.5061/dryad.2280gb65r)

## Description of the data and file structure

This dataset contains the North American golden eagle genome assembly liftover annotation as well as accompanying code for generating the annotation and analytical code for comparing assembly quality to other Accipitridae assemblies.

### Data Files

#### 1. bAquChr2.0.hap1.liftover_polished.gtf

**Description:** liftover annotation of the assembly using the European golden eagle reference genome (bAquChr1.4) annotation; generated by the script **run_liftoff.sbatch**

### Analysis Script Files

#### 1. CCGP_busco.sbatch

**Description:** code for performing the BUSCO analysis; [slurm](https://slurm.schedmd.com/man_index.html) submission script

#### 2. run_minimap2.sh

**Description:** code for aligning the assembly to the European golden eagle reference genome ([bAquChr1.4](https://www.ncbi.nlm.nih.gov/datasets/genome/GCF_900496995.4/)) for synteny assessment before performing liftover annotation (**run_liftoff.sbatch**); catenate script and execute each command on the command line

#### 3. run_liftoff.sbatch

**Description:** code for generating the liftover annotation; [slurm](https://slurm.schedmd.com/man_index.html) submission script

## Code/software Requirements

**CCGP_busco.sbatch**

Dependencies: [BUSCO](https://busco.ezlab.org/) v5.0.0+

Inputs:

* aves_odb10 -- ortholog database; download to working directory using `busco download -l aves_odb10`
* directory containing all genome assemblies for generating BUSCO scores
  * [bAquChr2.0.hap1](https://www.ncbi.nlm.nih.gov/datasets/genome/GCA_030820615.1/) -- North American golden eagle (CCGP assembly)
  * [Aquila_chrysaetos-1.0.2](https://www.ncbi.nlm.nih.gov/datasets/genome/GCF_000766835.1/) -- North American golden eagle (2014 Van Den Bussche assembly)
  * [AquilaChrysaetos1](https://www.ncbi.nlm.nih.gov/datasets/genome/GCA_000696035.1/) -- North American golden eagle (2014 Doyle assembly)
  * [bAquChr1.4](https://www.ncbi.nlm.nih.gov/datasets/genome/GCF_900496995.4/) -- European golden eagle
  * [Ajapo_1](https://www.ncbi.nlm.nih.gov/datasets/genome/GCA_030162315.1/) -- Japanese golden eagle
  * [bHalAlb1.1](https://www.ncbi.nlm.nih.gov/datasets/genome/GCF_947461875.1/) -- White-tailed eagle
  * [bGypBar2.pri](https://www.ncbi.nlm.nih.gov/datasets/genome/GCA_028022735.1/) -- Bearded vulture
  * [bHarHar1_primary_haplotype](https://www.ncbi.nlm.nih.gov/datasets/genome/GCF_026419915.1/) -- Harpy eagle
  * [ASM2749777v1](https://www.ncbi.nlm.nih.gov/datasets/genome/GCA_027497775.1/) -- Black-mantled goshawk
